# Pigsty
为中粮集团开发的猪舍进销管理系统。

一、系统架构
  1、后端：spring+springmvc+mybatis+druid;
  2、前端：jquery easyui+bootstrap;
  3、数据库：mysql；
  4、集成第三方组件：finereport,echarts等。
  
二、常用功能
  1、 常用字典添加、修改、查询与删除；
  2、 猪舍分配管理；
  3、 猪舍批量进销管理，支持可编辑表格处理，打印，导出等功能；
  4、 统计分析：支持finreport查询与统计，echarts常用图表；
  5、 系统管理：密码修改，退出系统等常用功能。
